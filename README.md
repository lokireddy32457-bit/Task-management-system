# Task Management System

This is a Flask-based Task Management System built for the Python Development Internship assignment. It allows users to register, log in, and manage their tasks efficiently.

## Technologies Used
* **Backend:** Python, Flask
* **Database:** postgreSQL 
* **Real-time/Async:** Flask-SocketIO (Eventlet)
